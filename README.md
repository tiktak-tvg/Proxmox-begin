## Proxmox
Виртуализация  Proxmox

### Установка Proxmox
Proxmox VE предъявляет следующие минимальные требования к серверу:

- Процессор: 64-разрядный (x86-64), совместимый с Intel VT-x или AMD-V для аппаратной виртуализации.

- Оперативная память: минимум 1 ГБ, но на практике рекомендуется от 4-8 ГБ для комфортной работы виртуальных машин.

- Дисковое пространство: минимум 16 ГБ для системы плюс дополнительное место для виртуальных машин.

- Сеть: как минимум одна сетевая карта (NIC).

- Дополнительно: Для использования ZFS или Ceph требуется как минимум дополнительно 8 ГБ ОЗУ.

#### Скачивание ISO-образа

Действие: Скачайте последнюю версию ISO-образа Proxmox VE с официального сайта.

Нюансы: Убедитесь, что загружаете стабильную версию.

#### Создание загрузочной флешки
После скачивания ISO-образа вам нужно записать его на USB-накопитель.

##### На Windows:
Используйте бесплатные утилиты, такие как Rufus или Etcher для записи ISO-образа на USB-накопитель. Инструкции по использованию этих программ обычно просты и интуитивно понятны.

##### На Linux:
Вы можете использовать утилиту командной строки dd.

Определите имя USB-накопителя: Выполните команду ``lsblk``, чтобы найти нужное устройство (например, ``/dev/sdb``). Будьте внимательны, чтобы не ошибиться с диском!.

Размонтируйте устройство: Если оно автоматически смонтировано, выполните ``sudo umount /dev/sdX1`` (замените sdX1 на вашу секцию).

Запишите образ: Выполните команду:

```bash
sudo dd bs=1M conv=fdatasync if=путь/к/файлу.iso of=/dev/sdX status=progress
```
(Замените путь/к/файлу.iso на путь к вашему ISO-файлу, а /dev/sdX — на имя USB-устройства).

#### Процесс установки с ISO-образа
Это основной и рекомендуемый метод для большинства пользователей.

Загрузка: Вставьте подготовленный USB-накопитель в сервер, загрузитесь с него (возможно, потребуется изменить порядок загрузки в BIOS/UEFI). Отключите Secure Boot для версий старше 8.1.

Запуск установки: В появившемся меню выберите "Install Proxmox VE" (обычно графический вариант). Появится окно приветствия и лицензионное соглашение (EULA), которое нужно принять.

Выбор диска: Выберите жесткий диск (или SSD), на который будет установлена система. Рекомендуется использовать файловую систему ZFS для расширенных функций (например, снимки), но можно выбрать и ext4 для простоты. При выборе ZFS, если у вас несколько дисков, их можно сразу объединить в RAID.

Региональные настройки: Укажите вашу Страну, Часовой пояс и Раскладку клавиатуры.

Настройка пароля: Установите надежный пароль для суперпользователя root и введите ваш Email для уведомлений.

#### Сетевая конфигурация: Самая важная часть. Вам нужно задать:

- Сетевой интерфейс: Выберите нужный.

- Имя хоста (Hostname): Например, pve1.mydomain.local.

- IP-адрес: Статический IP для вашего сервера Proxmox.

- Маска подсети (например, 255.255.255.0).

- Шлюз (Gateway): IP-адрес вашего роутера.

- DNS-сервер: Обычно IP роутера или публичные DNS, например, 8.8.8.8.

Установка: Нажмите Install и дождитесь завершения процесса. 

По завершении сервер перезагрузится. 

После перезагрузки вы увидите приглашение входа в консоль. 

Вы также сможете подключиться к веб-интерфейсу, открыв в браузере адрес ``https://<IP_вашего_сервера>:8006``
